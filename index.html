DebaterHub Research Page

An interactive platform for middle and highschool level students to research and prepare for debates.

Features

The user is able to access the documents and passages from the research database in an external source that is verified as scholarly and credible.

This is an interactive 3D platform built with Three.js where students can look through search results represented as planets in an immersive space. A navigational sidebar displays data returned by the database query.

The hierarchical structure of the returned data (clusters→ documents→ sections→ passages) may be explored using click-and-drag navigation. Back button functionality re-generates the objects of the previous scene. A stack data structure was used to track the user’s movement through the levels of the returned data.

Passages may be selected within the Three.js scene. These passages are saved, persisted in the user’s browser, and displayed on the UI sidebar. Here, the user may delselect passages or display their full text upon click.

When the user clicks an object, information (e.g. data type, document link, title) is displayed in a popup sidebar and next-level elements are generated on the scene, with GSAP animations providing smooth transitions.
